# Правила от проекта MetaCubeX

Этот каталог содержит ссылки на правила, предоставленные проектом **[MetaCubeX/meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat)**.

## Благодарность

Огромное спасибо проекту **MetaCubeX** за разработку и поддержку высококачественных правил для гео-IP и гео-сайтов, а также за развитие ядра **Mihomo (Clash.Meta)**.

*   **Профиль проекта**: [MetaCubeX на GitHub](https://github.com/MetaCubeX)
*   **Репозиторий с правилами**: [meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat)
*   **Используемые ресурсы**: GeoIP (например, для `telegram`) и GeoSite правила.

## Использование

В данном проекте правила MetaCubeX используются **по ссылке** через публичный CDN (jsDelivr). Это означает, что файлы не копируются, а в конфигурациях указываются прямые URL-адреса, предоставленные автором:

`https://cdn.jsdelivr.net/gh/MetaCubeX/meta-rules-dat@meta/geo/geoip/telegram.mrs`

Использование по ссылке является разрешённым и соответствует лицензии GPL-3.0, так как не является распространением (redistribution) исходных файлов.
