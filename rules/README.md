# Правила для сервисов

В этой папке находятся файлы правил, взятые из открытых источников.
Все права принадлежат их авторам.

## 📚 Используемые правила

| Сервис | Автор | Лицензия | Репозиторий |
|--------|-------|----------|-------------|
| YouTube, Discord и другие | Nellimonix | GPL-3.0 | [ClashXRule](https://github.com/Nellimonix/ClashXRule) |
| Telegram (GeoIP) и другие | MetaCubeX | GPL-3.0 | [meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat) |
| Telegram (GeoSite) и другие | hydraponique | GPL-3.0 | [roscomvpn-geoip](https://github.com/hydraponique/roscomvpn-geoip) |
| CDN (Cloudflare, Amazon, Fastly, Akamai, cdn77, Oracle) | PentiumB | Не указана  | [CDN-RuleSet](https://github.com/PentiumB/CDN-RuleSet) |

## ⚖️ Лицензии

- Файлы от **Nellimonix** распространяются под лицензией **GPL-3.0**
- Файлы от **MetaCubeX** и **hydraponique** распространяются под лицензией **GPL-3.0**
- Файлы от **PentiumB** распространяются на условиях **стандартного авторского права**
