## ⚡ КОНСТРУКТОР КОНФИГОВ ДЛЯ CLASH / MIHOMO ⚡

Простой и удобный конструктор конфигов — всё в одном файле.

---

### 🌐 Сайт

👉 [**config-constructor**](https://BloodWolfik.github.io/config-constructor/)

---

### 🚀 Как пользоваться

1. **Загрузите ваш конфиг** — нажмите на область загрузки или перетащите файл `.yaml`/`.yml`
2. **Выберите прокси** — отметьте нужные сервера для основной и дополнительных групп
3. **Выберите правила** — отметьте сервисы, которые хотите направлять через прокси
4. **Настройте опции** — исключения для LAN, торрентов, NVIDIA
5. **Скачайте готовый конфиг** — нажмите «СОБРАТЬ КОНФИГ»

---

### ⚙️ Возможности

- ✅ Загрузка вашего конфига с сохранением всех настроек
- ✅ Создание неограниченного количества прокси-групп
- ✅ Выбор индивидуальных прокси для каждой группы
- ✅ Назначение правил на конкретные группы
- ✅ Автоматическое разрешение конфликтов (если выбраны одинаковые правила от разных авторов)
- ✅ Исключения для LAN, торрентов и NVIDIA
- ✅ Иконки для групп (свой URL или из встроенной библиотеки)
- ✅ Удобная навигация — всё сворачивается и разворачивается
